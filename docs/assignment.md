## Deskripsi

Buat sebuah **Software as a Service (SaaS) berbasis web** berdasarkan studi kasus nyata yang menyelesaikan masalah tertentu.

## Ketentuan

| No  | Ketentuan                                                                                                                     |
| --- | ----------------------------------------------------------------------------------------------------------------------------- |
| 1   | Website berdasarkan **studi kasus nyata** dan menyelesaikan masalah tertentu                                                  |
| 2   | Semua fitur terbuka untuk kebutuhan penilaian                                                                                 |
| 3   | Boleh pilih **bahasa pemrograman web** apa saja                                                                               |
| 4   | Website harus bisa **diakses online** (deploy lewat Vercel, Netlify, Cloudflare Worker, dll)                                  |
| 5   | **Wajib pakai database** (Supabase, Neon db, Cloudflare D1, atau lainnya)                                                     |
| 6   | Kumpulkan **dokumentasi PDF** berisi gambaran program + URL                                                                   |
| 7   | Sertakan **video presentasi** (upload YouTube unlisted / Google Drive), durasi **maks 10 menit**, wajib **menampilkan wajah** |
